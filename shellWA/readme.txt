Persyaratan:
-Python 3.x.x
-selenium terinstall.
-Geckodriver terinstall dan pathnya diletakkan yang benar sesuai OS.
-Mozilla Firefox

Cara Kerja:
1. Run script botWA.py
2. Pastikan izin eksekusi running script botWA.py
3. Bot otomatis membuka web.whatsapp.com kemudian lakukan scan secara manual.
4. Jika sudah, coba lakukan dengan mulai kirim chat '/wawan'. Maka Bot Otomatis terpanggil.
5. Kemudian lakukan perintah shell pertama dengan menampilkan list directory. 'cd' untuk Windows dan 'ls' untuk Linux
	Yaitu dengan cara menambahkan double forward slash '//cd'.
6. Jika mau keluar maka pastikan kirim chat '/keluar'. Di tempat dimana itu terakhir kali digunakan.
7. Maka bot dalam status idle dan dapat dipanggil lagi suatu saat.
